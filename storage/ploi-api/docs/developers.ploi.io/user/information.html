Get your own user information

GET api/user

Example request

curl -X GET "https://ploi.io/api/user" \
-H "Authorization: Bearer {token}" \
-H "Content-Type: application/json" \
-H "Accept: application/json"

Example response

HTTP/1.1 200 OK
Content-Type: application/json
