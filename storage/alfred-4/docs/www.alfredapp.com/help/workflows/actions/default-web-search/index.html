Default Web Search Action

This is a super-handy action! Open your {query} in one of Alfred's default web searches.

Example using the Default Web Search action

I used one of the workflow templates to create this YouTube s earch. The template is the "Open default web search in specified browser" from the Web and URLs list.

I set the keyword to "yt" and specified that I want to open the YouTube default web search in Chrome.

To use a workflow like this, click on the + at the bottom of your workflows list, and select Templates > Web and URLs > Open Default Web Search in Specified Browser. This will launch a new workflow based on the same template.
